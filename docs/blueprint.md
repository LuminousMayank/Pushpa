# **App Name**: Pushpa Global Sourcing

## Core Features:

- Hero Section with Client Marquee: Visually striking hero section with a headline, subline, CTAs, and a rolling client logo marquee to establish trust and showcase major clients.
- Industry-Specific Sections: Dedicated sections for Defence & Marine, Steel Plants, and Cement & Power, each highlighting relevant supplies and linking to related products.
- RFQ Form with Intelligent Assistance: Contact form with an LLM that suggests the optimal wording and completeness check. When possible and useful, the LLM will automatically add technical clarifications based on available information such as sector and files uploaded to make the form more informative to the recipient. Serves as a tool to enhance user submissions and reduce ambiguities, by suggesting changes that ensure clarity in their requirements.
- Product and Services Showcase: Interactive product and service showcase using cards/tiles and accordions to present information clearly and concisely.
- About Page with Timeline: Compelling 'About Us' section with a timeline or milestones to showcase the company's history, values, and differentiators.
- Client Section: Showcase trust by displaying leading Indian clients, logos grouped by sector.

## Style Guidelines:

- Apple-style spacing and layout for a clean and spacious design.
- Sticky top navigation bar with logo on the left, menu in the center/right, and a highlighted 'Request a Quote' button.
- Body and headline font: 'Inter' for a clean, modern, and readable sans-serif experience. This sans-serif will ensure readability across devices. Note: currently only Google Fonts are supported.
- Smooth fade-in on scroll and hover states, creating a modern feel.
- Primary color: Deep blue (#1A237E) to convey trust, reliability, and corporate strength.
- Background color: Dark gray (#263238) provides a modern and sophisticated backdrop.
- Accent color: Gold/Orange (#FFB300) used sparingly to highlight key CTAs and create visual interest. Use only for important things.
